
# Barter my Book API

> A platform for trading books seamlessly through an intuitive and versatile API
> 
This project runs with Laravel version  ^10.10.

## Getting started

Assuming you've already installed on your machine: PHP (>= 8.0.0), [Laravel](https://laravel.com), [Composer](https://getcomposer.org) .

``` bash
# install dependencies
composer install

# create .env file and generate the application key
cp .env.example .env
php artisan key:generate



Then launch the server:

``` bash
php artisan serve
```

The  project is now up and running! Access it at http://localhost:8000.
